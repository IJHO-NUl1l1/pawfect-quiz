# 퀴즈 문항 초안 v4 (2026-07-08)

- **14문항**, 문항당 선택지 4개. 점수 매핑은 다음 과제 — 여기서는 문장·순서만 확정한다.
- 주제 구조는 IAMS Breed Selector(14문항)·AKC Breed Selector의 검증된 커버리지, 문장 톤은 한국 바이럴 테스트풍 일상 문장.
- 선택지 배열은 **라이트 → 헤비 순**, 문항 배치는 **변별 weight 오름차순** (CLAUDE.md 문항 구성 원칙): 앞부분 = 가볍고 재미있는 소프트 신호(성격·취향), 후반부 = 견종 구분을 강하게 가르는 문항, 피날레 = 크기.
- 각 문항의 `참고`는 주제 출처, `feature`는 이 문항이 신호를 줄 것으로 의도한 특징(delta 설계는 다음 과제).
- 실제 앱 데이터: `src/data/questions.ts` (이 문서와 수동 동기화 — 수정 시 양쪽 다 갱신할 것)

## 전반부 — 소프트 신호 (성격·라이프스타일)

### Q1. 어떤 성격의 강아지와 함께하고 싶나요?

1. 겁이 많고 소심해도 주인만 바라보는 조용한 껌딱지 강아지
2. 모든 사람들의 사랑을 독차지 하고 싶어하는 사람 좋아 강아지
3. 용맹하고 든든한 경찰견 or 보디가드 강아지
4. 하루 10km 달리기도 거뜬한 어질리티 천재 운동 천재 강아지

> 참고: 사용자 원안(2026-07-08), IAMS Q6 / feature: protective, stranger_friendly, energy_level, trainability, independence, intensity (다중 소프트 신호 — 재미 훅으로 1번 배치)

### Q2. 강아지와 나, 어떤 사이였으면 좋겠어요?

1. 24시간 껌딱지 — 화장실까지 따라와도 행복해요
2. 껌딱지까진 아니어도 늘 같은 방에 붙어 있는 사이
3. 각자 할 일 하다가 만나는 쿨한 룸메이트
4. 호기심 넘치는 모험가 — 새 냄새만 나면 탐험 출발!

> 참고: 성격 축 보강(DogTime wanderlust) / feature: independence, tolerates_alone

### Q3. 강아지의 텐션은 어느 정도가 좋아요?

1. 고요하고 잔잔하게 — 눈빛으로 교감하는 스타일
2. 꼬리 살랑살랑 — 은은하게 표현하는 정도
3. 퇴근하면 신나서 뱅글뱅글 환영 파티
4. 온몸으로 돌진하는 저세상 텐션도 다 받아줄게요

> 참고: 성격 축 보강(DogTime intensity) / feature: intensity, energy_level

### Q4. 주말 아침, 눈 뜨자마자 하고 싶은 건?

1. 이불 속에서 밀린 드라마 정주행 — 주말은 쉬는 날이죠
2. 동네 카페까지 슬렁슬렁 산책 정도?
3. 공원에서 가볍게 러닝이나 자전거 타기
4. 등산이든 한강 10km든 — 몸을 써야 진짜 주말이죠

> 참고: IAMS Q8(활동), AKC(에너지) / feature: energy_level

### Q5. 평일 낮, 우리 집 풍경은?

1. 야근에 약속에… 집을 자주, 오래 비워요
2. 아침에 나가면 저녁에야 돌아와요
3. 반나절 정도만 비우고 금방 돌아와요
4. 재택근무라 거의 항상 함께 있어줄 수 있어요

> 참고: IAMS Q9(혼자 있는 시간) / feature: tolerates_alone

### Q6. 강아지와 살아본 적 있나요?

1. 완전 처음이라 설렘 반 걱정 반이에요
2. 처음이지만 영상 보면서 공부 많이 했어요
3. 한 번 키워봐서 어느 정도는 알아요
4. 어릴 때부터 쭉 — 강아지 없는 집을 상상 못 해요

> 참고: IAMS Q4(소유 경험) / feature: novice_friendly

### Q7. 강아지 교육, 어떻게 시킬 생각이에요?

1. 순한 아이 만나서 자연스럽게 지내고 싶어요
2. 배변이랑 기본 예절만 되면 충분해요
3. '앉아'부터 하이파이브까지 — 같이 배우는 게 재밌을 것 같아요
4. 훈련사님 수업까지 들으면서 제대로 가르쳐볼 거예요

> 참고: IAMS Q10(훈련 정도) / feature: trainability, novice_friendly

### Q8. 간식 달라고 눈망울 공격이 들어왔다!

1. 그 눈빛을 어떻게 이겨요… 달라는 대로 주게 될 것 같아요
2. 그때그때 기분 따라 유연하게 간식 타임
3. 간식은 훈련 성공했을 때 보상으로만
4. 정량 급여에 수제 간식까지 — 식단 관리 제대로 할 거예요

> 참고: 자체 문항 (weight_gain 커버용) / feature: weight_gain, trainability

## 후반부 — 변별 weight 큰 문항 (견종 구분 강함)

### Q9. 현관 벨이 울렸어요. 강아지가 어떻게 반응하면 좋겠어요?

1. 짖는 건 곤란해요 — 우리 집은 방음이 약하거든요
2. 무심하게 내 옆에 딱 붙어 있기 — 세상 조용하게
3. 꼬리 흔들며 손님을 제일 먼저 반기는 인싸견이었으면
4. 누가 왔는지 우렁차게 알려주는 믿음직한 경비 알람이 됐으면

> 참고: IAMS Q6·Q7(보호 성향+짖음) / feature: barking, stranger_friendly, protective

### Q10. 산책은 어떤 스타일이 될 것 같아요?

1. 솔직히 자주는 못 해줄 것 같아요 — 실내 놀이 위주로
2. 날씨 좋은 날 골라서 가볍게 다녀올게요
3. 아침저녁으로 꼬박꼬박, 루틴대로 나가요
4. 어질리티든 등산이든 — 눈밭이든 빗속이든 뭐든 같이해요

> 참고: AKC(운동/날씨), IAMS Q8, 사용자 지시 예시 / feature: energy_level, cold_tolerance

### Q11. 강아지 털에 대한 솔직한 마음은?

1. 짧고 털빠짐 적은 깔끔한 스타일이 좋아요 — 청소는 최소로
2. 어떤 털이든 좋은데, 옷에 붙는 건 좀 곤란해요
3. 복슬복슬 장모도 환영 — 빗질쯤이야 힐링이죠
4. 길든 많이 빠지든 상관없어요 — 미용실 비용도 아낌없이 쓸게요

> 참고: IAMS Q12·Q13, 사용자 지시 예시 / feature: coat_length(std 1.50), shedding, grooming_ease

### Q12. 지금 집에는 누구누구 살아요?

1. 나 혼자 — 강아지가 첫 식구가 될 거예요
2. 고양이(또는 햄스터·새)가 먼저 자리 잡고 있어요
3. 이미 강아지가 한 마리 — 둘째 계획 중이에요
4. 아이까지 있는 북적북적 대가족이에요

> 참고: IAMS Q3·Q5 / feature: kid_friendly, prey_drive, dog_friendly, tolerates_alone (조건부 강신호)

### Q13. 지금 살고 있는 집은 어떤 곳인가요?

1. 벽이 얇아서 층간소음이 걱정되는 집이에요 — 조용해야 해요
2. 나 하나로도 꽉 차는 아늑한 원룸이에요
3. 방이 여러 개인 널찍한 아파트예요
4. 마당 있는 단독주택 — 대형견도 마음껏 뛰놀 수 있어요

> 참고: IAMS Q1·Q2, 노트펫(소음) / feature: apartment_friendly(std 1.54 — 전체 1위), size, barking

### Q14. 강아지를 품에 안는 상상을 해보세요. 어떤 느낌이 좋아요?

1. 한 손에 쏙 들어오는 콩알만 한 아기
2. 무릎 위에 딱 맞는 사이즈
3. 안으면 팔이 꽉 차는 든든한 중형견
4. 오히려 걔가 날 안아줄 것 같은 대형견

> 참고: IAMS Q11(크기 선호) / feature: size (가장 강한 단일 구분 — 피날레 배치)
